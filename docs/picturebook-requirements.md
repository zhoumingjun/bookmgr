# 线上绘本库 - 原始需求

> 录入时间：2026-03-31
> 来源：线上绘本库.docx

## 产品定位

特教（培智）新生绘本资源管理平台，围绕幼小衔接五大适应维度构建，适配教师（PC端）和家长（手机端）。

整体系统操作简洁、界面视觉友好，适配教师电脑端、家长手机端操作；无复杂操作流程，按钮、文字放大显示，适配特教人群使用的视觉、操作习惯；所有功能模块围绕五大适应维度展开，资源分类清晰、检索高效。

---

## 一、用户角色及权限

| 角色 | 说明 | 核心权限 |
|------|------|----------|
| 超级管理员 | 课题负责人 | 全量权限：用户增删改查、绘本资源全量操作、数据统计、系统设置等 |
| 普通管理员 | 课题核心成员 | 绘本上传、编辑、审核，家长/教师用户信息查看，资源使用数据查看 |
| 教师用户 | — | 检索、查看、下载所有绘本，查看配套教学资料，收藏常用绘本，提交使用反馈 |
| 家长用户 | — | 仅可查看、浏览课题组推送的绘本及亲子共读指导资料，收藏绘本，提交阅读反馈 |

### ⚠️ 用户管理规则
- **所有用户由超级管理员创建并分配角色，禁止自主注册**
- 超级管理员手动创建用户，分配角色后告知初始密码或强制首次修改密码
- 关闭自主注册功能

---

## 二、绘本资源分类（核心）

按新生入学适应五大维度进行一级分类，每个一级分类下可根据教学场景设置二级子分类，分类结构可由管理员灵活调整。

### 初始分类结构

| 一级分类 | 二级子分类 |
|----------|-----------|
| 身心准备 | 校园作息适应、身体动作发展、校园规则感知 |
| 生活准备 | 穿衣、洗漱、就餐、整理物品、如厕等校园生活技能 |
| 学习准备 | 课堂常规、文具使用、注意力培养、简单课堂互动 |
| 社会准备 | 与同学交往、与老师沟通、集体活动参与、分享合作 |
| 情绪准备 | 入学焦虑缓解、情绪识别、情绪表达、挫折应对 |

## 三、绘本资源上传与编辑

### 3.1 纸电双版适配
- 绘本电子版：PDF、图片集、音频绘本、动画绘本
- 纸质绘本扫描件/实拍图

### 3.2 标准化信息表单字段

| 字段名 | 说明 |
|--------|------|
| 绘本名称 | — |
| 所属适应维度 | 一级 + 二级分类 |
| 核心适应目标 | — |
| 适配培智新生障碍类型/认知水平 | — |
| 绘本页数/时长 | — |
| 教学使用建议 | — |
| 亲子共读指导 | — |
| 资源类型 | 纸版 / 电子版 / 音频 / 动画 |
| 上传人 | — |
| 上传时间 | — |

### 3.3 审核机制
- 绘本资源上传后需经超级管理员/指定审核人审核通过后，方可在前台展示

### 3.4 批量操作
- 支持批量上传绘本资源
- 支持批量编辑绘本所属分类、核心目标等共性信息

## 四、绘本资源展示

### 4.1 卡片式呈现
- 卡片包含：绘本封面图、绘本名称、所属核心维度、适配认知水平
- 封面图、文字放大显示
- 色彩对比明显

### 4.2 详情页
- 展示绘本所有标准化信息
- 在线查看、下载（教师用户）、在线播放（音频/动画绘本）
- 纸质绘本提供高清实拍图翻页查看功能
- 单独设置教学/阅读指导板块（支持复制、下载）

---

## 五、检索与筛选

### 5.1 多维度检索
- 关键词检索（绘本名称、核心目标）
- 维度筛选（单选/多选五大适应维度）
- 资源类型筛选（电子版/音频）
- 检索结果实时展示，无延迟

### 5.2 快速检索
- 首页设置五大维度快速入口
- 点击直接进入对应维度的绘本列表，无需二次筛选

### 5.3 检索结果排序
- 按上传时间排序
- 按使用热度排序
- **默认按使用热度排序**，优先展示高频使用的优质绘本

## 六、附录：术语说明

- **特教/培智**：特殊教育/培智学校，面向智力障碍学生的教育
- **五大适应维度**：幼小衔接的核心适应领域
- **纸电双版**：纸质绘本 + 电子版本
- **认知水平**：培智学校学生按障碍类型/认知水平分级
